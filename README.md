GitHub Sınıf Katılım Etkinliği

Bu repository GitHub kullanımını öğrenmek amacıyla oluşturulmuştur.
Aşağıdaki adımları sırayla uygulayın.

1) Bu repoyu kendi hesabına kopyala (Fork)
Sayfanın sağ üst köşesindeki "Fork" butonuna bas.
Repo artık senin hesabına kopyalanmış olacak.

2) README dosyasını aç ve ismini ekle
Kendi hesabına kopyalanan repoda README.md dosyasına gir.
Sağ üstteki düzenleme (kalem) ikonuna bas.
Sayfanın en altına şu formatta ismini yaz:

- Ad Soyad / Bölüm

Örnek:
- Ahmet Yılmaz / Yönetim Bilişim Sistemleri

3) Değişikliği kaydet (Commit)
Sayfanın en altına in.
Commit message alanına şunu yaz:

İsim eklendi

Sonra "Commit changes" butonuna bas.

4) Katkını gönder (Pull Request)
Committen sonra görünen "Contribute" butonuna bas.
"Open pull request" seçeneğine tıkla.
Ardından "Create pull request" butonuna bas.

İsmin ana projeye gönderilmiş olacak.

Not:
Sadece kendi ismini ekle, başkasının ismini silme veya düzenleme.

Tebrikler!
GitHub üzerinde ilk Fork, Commit ve Pull Request işlemini yaptın.
